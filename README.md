# Email-SMS-Spam-Classifier

A Machine Learning project that classifies messages as **Spam** or **Ham (Not Spam)** using Natural Language Processing (NLP) techniques.

## 🚀 Project Overview

Spam messages are a common problem in email and SMS communication. This project uses **Machine Learning and NLP techniques** to automatically detect whether a message is spam or not.

The model is trained on a dataset of labeled SMS messages and learns patterns in text to classify incoming messages.

## 🧠 Machine Learning Workflow

1. Data Collection
2. Data Cleaning
3. Text Preprocessing
4. Feature Extraction (TF-IDF / Count Vectorizer)
5. Model Training
6. Model Evaluation
7. Spam Prediction
